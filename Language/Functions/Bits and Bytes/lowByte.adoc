= lowByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrizione
Extracts the low-order (rightmost) byte of a variable (e.g. a word).
[%hardbreaks]


[float]
=== Sintassi
`lowByte(x)`


[float]
=== Parametri
`x`: a value of any type

[float]
=== Restituisce
byte
