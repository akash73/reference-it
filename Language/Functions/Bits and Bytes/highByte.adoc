= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrizione
Extracts the high-order (leftmost) byte of a word (or the second lowest byte of a larger data type).
[%hardbreaks]


[float]
=== Sintassi
`highByte(x)`


[float]
=== Parametri
`x`: a value of any type

[float]
=== Restituisce
byte
